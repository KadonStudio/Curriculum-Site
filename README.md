# Thumbprint Curriculum — Website

Static multi-page website for Thumbprint Curriculum, built with HTML, CSS, and vanilla JavaScript.

## Pages

| File | Page |
|---|---|
| `index.html` | Home — hero, social carousel, stats |
| `about.html` | About — story, meet the team, process |
| `products.html` | Products — all 4 product overviews + pricing |
| `contact.html` | Contact — message form |
| `get-started.html` | Get Started — MagnaCon booking + general inquiry |
| `signature.html` | Thumbprint Signature — avatar creation checkout |
| `amplify.html` | Thumbprint Amplify — subscription checkout |
| `circle.html` | Thumbprint Circle — community membership checkout |
| `renew.html` | Thumbprint Renew — discovery call request |

## Serving Locally

```bash
node serve.mjs
```

Then open [http://localhost:3000](http://localhost:3000).

## Brand

- **Colors:** `#22AEE4` (bright blue), `#0F6EB7` (dark blue), `#2E2E36` (dark), `#000000` (black)
- **Font:** Montserrat (Google Fonts)
- **Shared styles:** `style.css`
